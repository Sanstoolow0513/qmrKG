
# QMR的毕业设计

## 题目介绍

基于大模型的课程知识命名实体识别与关系抽取

### 课题内容

- 课程语料构建与预处理：收集课程教学大纲、教材章节、课堂讲义、实验指导书及历年试题等多源文本材料；开展课程语料标准化处理与分层标注，形成用于大模型抽取的结构化输入语料库。
- 课程知识命名实体识别模型研究： 设计基于大模型的课程命名实体识别方法，针对"协议名称"、"概念术语"、"机制算法"、"性能指标"等不同类别进行标签定义与示例标注；研究零样本（zero-shot）与少样本（few-shot）识别的提示工程（Prompt Engineering）策略，提高在课程领域内的实体识别准确率与泛化性。
- 课程知识关系抽取与语义表示学习 构建适用于课程语料的关系类型体系（如"包含关系"、"依赖关系"、"对比关系"、"应用关系"）；利用大模型的上下文推理能力，识别实体间显式与隐式关系；结合向量嵌入（Embedding）方法进行语义关联建模，形成多粒度知识关系网络。
- 基于大模型的知识抽取管线设计与实现 构建一条课程知识自动化抽取流程（Pipeline），包括：文本输入 → 实体识别 → 关系抽取 → 三元组生成 → 知识验证与融合；实现可复用的知识抽取系统原型，支持后续课程知识图谱构建与教学Agent调用。
- 应用验证与评估：在课程中进行示范验证，评估模型在知识覆盖率、抽取准确率、可解释性及人工干预成本等方面的表现；分析其在教学智能问答、作业生成与知识可视化等场景下的可行性

### 课题任务要求

本项目旨在探索将大语言模型LLM技术引入高校课程知识建模过程，研究并实现面向计算机网络等计算机类课程的知识命名实体识别（NER）与关系抽取（RE）方法体系。本项目以大模型的自然语言理解与语义推理能力为基础，建立一种能够自动识别课程核心知识点、提取概念关系并生成可复用知识图谱结构数据的技术路线
